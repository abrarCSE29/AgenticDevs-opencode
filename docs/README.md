# Project Documentation

This directory contains all project documentation generated by the agent swarm.

## Templates

- [BRD Template](templates/brd-template.md) — Business Requirements Document template
- [PRD Template](templates/prd-template.md) — Product Requirements Document template
- [Architecture Template](templates/architecture-template.md) — Architecture documentation template

## Document Naming Convention

| Document | Filename Pattern | Generated By |
|----------|-----------------|--------------|
| Business Requirements | `brd-<project-name>.md` | `@business-analyst` |
| Product Requirements | `prd-<project-name>.md` | `@business-analyst` |
| Architecture | `architecture-<project-name>.md` | `@solution-architect` |
| Best Practices | `best-practices-<project-name>.md` | `@solution-architect` |

## Agent Workflow

1. **Business Analyst** (`@business-analyst`) — Creates BRD and PRD from project descriptions
2. **Solution Architect** (`@solution-architect`) — Reads BRD/PRD, creates architecture and best practices docs

### Usage

From the primary Build or Plan agent:

```
@business-analyst Create a BRD and PRD for a new e-commerce platform
@solution-architect Read the BRD and PRD in docs/ and create the architecture document
```
